# 框架能力

## 学习目标

- 深入掌握主流前端框架的使用与原理
- 能结合项目经验回答框架设计与优化问题
- 能区分会用、会讲、会分析原理三个层次

## 文档导航

- Vue 面试复习清单：[Vue-面试复习清单.md](./Vue-面试复习清单.md)
- Vue 响应式原理 Demo：[vue-响应式原理-demo/README.md](./vue-响应式原理-demo/README.md)

## 当前整理策略

- 第一阶段先聚焦自己主用框架 Vue2 / Vue3
- 先把 Vue 核心、Router、状态管理、性能优化整理成完整问答版
- 后续如果需要，再补 React 和更偏工程化的框架专题

## 建议整理内容

- Vue 核心概念
- Vue2 / Vue3 差异
- 响应式原理
- 组件通信
- 生命周期
- 路由原理
- 状态管理
- 渲染机制
- 常见性能优化
- 项目中的框架落地经验

## 面试怎么准备

- 优先围绕自己主用框架深入准备
- 每个主题尽量结合真实项目场景来讲
- 原理题要能讲清“为什么这样设计”
- 版本差异题尽量说清“为什么 Vue3 要这样改”
