# 性能与安全

## 学习目标

- 能系统回答前端性能优化思路
- 能识别并说明常见前端安全问题
- 能结合项目经验体现优化意识和风险意识

## 文档导航

- 性能面试复习清单：[性能-面试复习清单.md](./性能-面试复习清单.md)

## 当前整理策略

- 本轮先单独整理性能题，安全相关后续再拆独立文档
- 性能文档统一按“问题 + 简答版 + 展开版 + 关键点 + 常见追问 + 追问答案 + 容易答偏的地方”整理
- 题目顺序优先按“方法论 -> 加载优化 -> 资源与构建 -> 渲染与运行时 -> 指标与监控”组织
- 后续如果内容继续扩展，再按加载性能、渲染性能、性能监控等专题拆分

## 建议整理内容

- 首屏优化
- 懒加载与预加载
- 资源压缩与缓存
- 图片优化
- 代码分割
- 性能指标
- XSS
- CSRF
- CSP
- 监控与异常上报

## 复习建议

- 性能题优先按“发现问题-分析原因-优化方案-验证结果”来讲
- 安全题优先按“是什么-怎么发生-怎么防”来讲
- 尽量整理真实项目中的优化案例
- 第一轮先过题目和简答版，先把主线说顺
- 第二轮重点补指标、工具、优化取舍和验证方法
- 第三轮尽量结合真实项目经历，把“做了什么”和“结果如何”串起来

## 当前优先级

1. 先吃透性能复习清单里的 14 道核心题
2. 优先掌握首屏优化、缓存、图片优化、代码分割、性能指标
3. 再把重绘回流、运行时优化、性能监控串成完整链路
4. 安全相关后续单独整理
