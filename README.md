# 前端面试复习仓库

用于整理前端面试相关的知识点、题目、答案、项目表达与练习代码。

这个仓库的目标不是单纯收藏资料，而是逐步沉淀成一套：

- 可复习
- 可检索
- 可复述
- 可持续补充

## 仓库目标

- 系统整理前端面试高频知识点
- 沉淀适合自己表达方式的答案
- 通过题目、代码和项目表述提高面试输出稳定性
- 形成长期可维护的个人复习资料库

## 当前复习重点

建议优先投入下面几个模块：

1. JavaScript 核心
2. 浏览器原理
3. 算法与手写题
4. 项目表达

这几个方向最容易同时影响八股题、手写题和项目深挖表现。

## 推荐复习顺序

如果你是按面试准备来使用这个仓库，建议按下面顺序推进：

1. `JavaScript 核心`
2. `浏览器原理`
3. `HTML 与 CSS`
4. `框架能力`
5. `工程化`
6. `性能与安全`
7. `算法与手写题`
8. `项目表达`

其中：

- `JavaScript 核心`、`浏览器原理`、`算法与手写题` 建议长期穿插复习
- `项目表达` 不建议放到最后才准备，应该尽早沉淀素材

## 使用方式

### 文档复习

- 先从各主题目录的 `README.md` 进入
- 再逐步补充和整理专题文档
- 每个知识点尽量补齐：概念、原理、示例、易错点、面试怎么说

### 算法与手写题练习

题目目录默认包含：

- `01-question.js`
- `02-answer.js`

运行方式：

```bash
node 01-question.js
node 02-answer.js
```
