# 输出示例

## 单题模式示例

目录：

```text
7-算法与手写题/01-算法题/001-两数之和/
├── 01-question.js
└── 02-answer.js
```

说明：

- 适用于“出一道题”“来一道题”
- 默认难度为中级
- 默认在无上下文时生成算法题
- `01-question.js` 顶部注释承载题面，并包含用户作答区和执行区域
- `02-answer.js` 顶部注释承载解题思路和复杂度说明，并包含最优解与执行区域

## 题单模式示例

目录：

```text
7-算法与手写题/02-手写题/
├── 001-实现防抖/
├── 002-实现节流/
├── 003-实现深拷贝/
├── 004-实现 Promise.all/
└── 005-实现并发控制/
```

说明：

- 适用于“来 5 题”“给我一套题”
- 每道题都使用 `01-question.js` 和 `02-answer.js`
- 题单中的难度应保持一致，除非用户明确要求混合难度
- 默认通过 `node 01-question.js` 和 `node 02-answer.js` 查看运行结果
