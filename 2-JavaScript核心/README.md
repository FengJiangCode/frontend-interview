# JavaScript 核心

## 学习目标

- 建立扎实的 JavaScript 语言基础
- 理解执行机制与核心运行模型
- 提升手写题与原理题回答能力

## 文档导航

- JavaScript 面试复习清单：[JavaScript-面试复习清单.md](./JavaScript-面试复习清单.md)

## 当前整理策略

- 先沉淀一份完整的 JavaScript 面试复习清单，作为统一入口
- 每道题统一按“问题 + 简答版 + 展开版 + 关键点 + 常见追问 + 追问答案 + 容易答偏的地方”整理
- 题目顺序优先按面试高频度和复习逻辑组织，先基础，再原型，再异步，再手写与工程化
- 后续如果内容继续扩展，再按专题拆分独立文档

## 建议整理内容

- 数据类型
- 作用域与作用域链
- 闭包
- `this`
- 原型与原型链
- 继承
- Promise
- `async/await`
- 事件循环
- 深拷贝
- 节流与防抖
- 模块化

## 复习建议

- 每个知识点同时准备定义、原理、示例
- 第一轮先过题目和简答版，确保能在 30 到 60 秒内说清楚
- 第二轮重点补展开版和常见追问，提升深挖题表现
- 第三轮结合手写题和机制类问题，训练流程化表达
- 高频手写题可以和对应原理题一起复习，避免知识点割裂

## 当前优先级

1. 先熟悉 JavaScript 复习清单里的 23 道核心高频题
2. 优先吃透 `this`、闭包、原型链、Promise、事件循环
3. 再把深拷贝、节流防抖、模块化和手写题串起来
4. 后续按专题继续拆分独立文档
