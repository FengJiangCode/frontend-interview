# AI 基础体系

## 学习目标

- 建立对 AI、大模型、Agent 生态的系统性认知
- 理解核心概念的职责边界和相互关系
- 能把抽象概念和真实工作场景对应起来

## 当前定位

- 这一部分偏 Agent 生态优先
- 不深入模型训练、微调、架构细节等偏底层内容
- 重点放在概念、关系、职责边界和实际协作方式

## 建议整理方式

- 每个主题尽量按“是什么、为什么需要、和什么容易混淆、工作中怎么用、面试怎么说”来整理
- 优先建立整体地图，再逐步细化到单个概念
