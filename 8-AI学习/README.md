# AI 学习

## 学习目标

- 系统理解 AI 基础体系，建立从大模型到 Agent 生态的整体认知
- 梳理 AI 在工作中的常见应用方式，形成可复用的提效方法
- 同时具备概念理解、场景判断和风险校验意识

## 当前整理策略

- 第一阶段先搭建完整框架，保证体系和应用两条线都能覆盖
- 基础体系优先聚焦 Agent 生态相关概念，不深入模型训练细节
- 工作应用优先整理高频、可落地、可复述的真实使用场景
- 先完成总览清单，后续再逐步拆成独立专题文档

## 建议整理内容

### 基础体系

- AI、大模型与 Agent 的关系
- Prompt、Context、Memory、Tool Use
- RAG、Workflow、Agent 的区别与联系
- Skill、MCP 在生态里的位置
- Agent、Skill、MCP 如何协作
- 常见误区与边界认知

### 工作应用

- AI 在研发工作流中的提效方式
- AI 在通用办公中的提效方式
- AI 在项目中的落地应用
- 输出校验、隐私与安全
- 项目表达和面试表达中的 AI 叙述方式

### Demo

- 可运行的教学 demo
- 概念演示和流程可视化
- 通过动手实验理解 Agent、Skill、MCP 的协作方式

## 面试怎么准备

- 不只记概念定义，要能讲清它们之间的关系和职责边界
- 不只说“我用过 AI”，要能讲清具体场景、收益、限制和校验方式
- 如果项目里用过 AI，尽量准备真实案例：为什么这样设计、为什么这样协作、最后效果如何
