# AI 工作应用

## 学习目标

- 梳理 AI 在真实工作中的高频应用方式
- 形成“提效 + 质量控制 + 风险意识”并重的使用习惯
- 能把实际使用经验整理成可复述的工作方法和项目表达

## 当前定位

- 第一版均分覆盖三类场景：研发、办公、项目落地
- 不只堆工具名称，重点整理使用方法、边界和校验方式
- 每个场景都尽量能落到真实工作动作上

## 建议整理方式

- 每个场景尽量按“能做什么、怎么协作、有什么风险、如何校验、怎么表达”来整理
- 优先沉淀自己真的会遇到、真的能复用的高频工作流
