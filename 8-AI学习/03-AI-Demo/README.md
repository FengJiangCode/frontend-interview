# AI Demo

## 目录说明

这个目录专门存放 AI 学习过程中的可运行 demo。

这些 demo 的目标不是追求完整产品功能，而是帮助理解：

- 核心概念
- 执行流程
- 协作关系
- 真实工作中的使用方式

## 当前 Demo
