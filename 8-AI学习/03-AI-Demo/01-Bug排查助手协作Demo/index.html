AI Demo / Realistic Mock Flow

01-Bug排查助手协作Demo

这不是“概念卡片”式演示，而是一条更贴近真实使用场景的调用链： 用户提需求，Agent 调度，LLM 做决策与总结，Skill 执行分析，MCP 提供外部资源。

本 Demo 的真实感边界

  • LLM 调用是 mock，但保留了 prompt、tool choice、summary 这些真实职责。
  • MCP 调用是 mock，但保留了资源列表、读取资源、搜索资源这些真实接口感。
  • Skill 是一个稳定任务能力，不直接替代 Agent 或 LLM。

Agent

接收目标、决定下一步、触发 LLM 与 Skill、整理最终结果。

LLM

负责判断该看什么、如何总结证据、如何组织输出。

Skill

封装 Bug 分析套路，保证流程和输出结构稳定。

MCP

暴露日志、源码、接口文档等资源给上层能力读取。
