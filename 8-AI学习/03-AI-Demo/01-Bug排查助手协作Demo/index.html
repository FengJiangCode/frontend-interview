01-Bug排查助手协作Demo

这版 demo 更贴近真实排障台：支持多个 bug case，能看到 Agent 的决策、LLM 的 mock 调用、Skill 的结构化输出，以及 MCP 的 tool call 参数。

怎么理解这版

  • LLM 是 mock，但会展示更像真实系统的 prompt 和响应。
  • MCP 是 mock，但会展示 list/read/search 这类工具调用入参。
  • 你可以切换不同 bug case，对比同一套协作链路如何处理不同问题。

Agent

接收目标、决定下一步、调度 LLM 与 Skill，并输出最终结论。

LLM

负责规划下一步、选择分析路径、基于证据总结结论。

Skill

封装稳定任务方法，例如日志分析、接口契约核对、缓存问题排查。

MCP

向上层暴露资源与工具，例如读取日志、读代码、搜索关键字。
