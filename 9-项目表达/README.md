# 项目表达

## 学习目标

- 能把项目经历讲得清楚、有重点、有说服力
- 能围绕个人贡献、技术难点和结果进行结构化表达
- 为项目深挖题和行为面试题准备稳定素材

## 建议整理内容

- 项目背景与目标
- 业务场景
- 技术选型
- 核心难点
- 优化方案
- 个人贡献
- 项目亮点
- 遇到的问题与复盘
- 指标结果

## 面试怎么准备

- 每个核心项目都准备 1 分钟版和 3 分钟版表达
- 优先按 STAR 或“背景-问题-方案-结果”来整理
- 不只讲做了什么，还要讲为什么这么做、效果怎么样
