## 简答版（1-2分钟）

这是滴滴国际化外卖面向巴西、墨西哥等6国运营团队的 CRM 工作平台，签约模块是商家入驻的核心节点。

核心挑战是：中后台表单极度复杂，单场景字段上百个，多国业务规则各不相同，旧代码硬编码严重，一有需求变更就牵一发动全身——产品想快速试错，但技术维护成本越来越高，形成恶性循环。

我的方案是引入"元数据平台 + Formily"组合：元数据平台让产品和运营可视化配置表单结构和规则，输出 JSON Schema；Formily 负责解析 JSON Schema 并渲染，彻底把表单逻辑和业务逻辑分开。同时基于 Formily 封装了 20+ 个业务组件，覆盖签约各环节高频控件。

重构后，新需求只需配置 JSON，平均开发周期从 5人天/需求 压缩到 1人天/需求，需求准时率提升至95%以上，代码风格一致率提升80%。

## 展开版（5分钟+）

### 背景与挑战

国际化外卖运营平台服务6国运营团队，签约模块是商家与平台达成合作的关键流程，涉及大量复杂表单（单场景字段100+，数据关联/逻辑联动/校验联动高度耦合）。项目经过多轮迭代后：

- 硬编码逻辑与业务强耦合，复用性极差
- 6国业务规则差异大，同一场景维护多套逻辑，代码膨胀严重
- 业务现状模糊，常出现"开发一半改需求"的情况

**核心矛盾：** 产品想快速试错，技术维护成本越来越高，两者形成恶性循环。

### 方案选型

**为什么不选纯组件化重构？**
组件化只解决了 UI 复用问题，业务逻辑依然散落在各页面组件里，多国规则差异问题没有本质改善，改需求时还是要改代码。

**为什么选"元数据平台 + Formily"？**
- **元数据平台**：让产品/运营直接配置表单结构和规则（JSON Schema），技术不再是唯一瓶颈，产品改配置不需要发版
- **Formily**：阿里开源的表单解决方案，原生支持 JSON Schema 驱动、字段联动、动态校验，恰好匹配我们的场景
- **结合效果**：业务规则变化 → 产品改配置 → 无需发版，彻底实现逻辑与页面解耦

### 关键实现

1. **主导 Formily 组件化开发**：基于 Formily 二次封装，抽象20+业务组件（地区选择、证件上传、费率配置等签约高频控件）
2. **负责签约场景迁移**：将10+核心签约流程逐步迁移到配置化体系，制定迁移规范和 JSON Schema 设计标准
3. **技术推广**：组内举办3次技术分享，编写最佳实践文档，帮助团队统一技术栈，代码风格一致率提升80%

## 高频追问 Q&A

**Q1：Formily 具体怎么实现字段联动的？**
Formily 有一套响应式状态管理（基于 `@formily/reactive`），字段之间通过 `reactions` 配置互相监听。比如"国家"字段变化时，触发"费率方案"字段的选项联动，全部通过 JSON Schema 描述，不写命令式代码。

**Q2：元数据平台是怎么与 Formily 结合的？**
元数据平台的职责是让非技术人员可视化配置表单，最终输出一份标准的 JSON Schema。Formily 的 `SchemaForm` 组件直接消费这个 JSON，解析并渲染出完整的表单 UI，包括校验规则和联动逻辑。我们做的核心工作是：定义 JSON Schema 规范 + 封装自定义业务组件 + 处理 Formily 与业务逻辑的边界。

**Q3：你负责重构，原有业务怎么保证稳定迁移？**
采用渐进式迁移策略。先选复杂度中等的签约流程做试点，新老方案双跑对比，确认功能对齐后再推进。同时制定了清晰的迁移规范，让其他同学也能独立完成迁移。

**Q4：团队接受新方案的阻力大吗？怎么推动的？**
有一定阻力，主要是 Formily 的学习成本。我的做法：先做出能跑的 Demo 降低感知门槛；再组织3次技术分享，结合实际业务案例讲，而不是照着文档念。最后用数据说话——第一个迁移场景上线后开发周期明显缩短，团队自然接受了。

**Q5：JSON Schema 的版本管理怎么做的？**
Schema 存在元数据平台，有版本记录和变更审计，不是散落在代码里的文件。表单渲染时拉取对应版本的 Schema，历史数据可回溯。如果 Schema 有破坏性变更，需要走评审流程。

## 容易答偏的地方

1. **不要把重点放在 Formily 的 API 细节上**。面试官更关心你为什么选它、怎么推动落地、带来什么业务价值。除非对方专门追问原理，否则不要主动展开 `reactive`、`SchemaField` 等具体 API。

2. **不要把"引入了配置化"说成银弹**。面试官可能追问边界：哪些场景仍需硬编码（高度定制化的交互）、JSON Schema 如何版本管理、如果产品配置出错怎么回滚。提前想好答案。
